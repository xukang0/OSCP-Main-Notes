## 01. Initial NMAP Enumeration

Compile a list of accessible hosts and add it to a `hosts.txt` file.

`-iL hosts.txt`

TCP Scan
```dataviewjs
const page = dv.page("Templater/IP");const ip = page?.IP ?? "NO IP FOUND";

const command = `sudo nmap -sV -Pn -T4 -oA initial_scan ${ip}`;

dv.paragraph("```bash\n" + command + "\n```");
```
UDP Scan
```dataviewjs
const page = dv.page("Templater/IP");const ip = page?.IP ?? "NO IP FOUND";

```

Check the [[0 Port Number Table]] and their associated services, run service scans on interesting ports.
```dataviewjs
const page = dv.page("Templater/IP");const ip = page?.IP ?? "NO IP FOUND";

const command = `sudo nmap -A -oA script_scan ${ip} -p [portno]`;

dv.paragraph("```bash\n" + command + "\n```");
```
Add hosts to /etc/hosts 
```dataviewjs
const page = dv.page("Templater/IP");const ip = page?.IP ?? "NO IP FOUND";

const command = `echo "${ip} [domain_name]" | sudo tee -a /etc/hosts`;

dv.paragraph("```bash\n" + command + "\n```");
```
